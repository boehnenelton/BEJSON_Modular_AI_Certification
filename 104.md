# BEJSON 104 Standard (Level 2 - Objects)
**Status:** Active | **Type:** Standalone Record
**Handling Certification:** LEVEL 2 (SPECIALIZED)

## 📋 Overview
The **104** standard is a Level 2 specialization building upon Level 1 fundamentals. It is used for standalone entities or complex objects (e.g., AI Profiles).

## 🎖 Level 2 Qualification Rule
Studying this documentation grants **Level 2 Qualification** for the **104** standard, provided **Level 1** is already mastered.

## 🚫 MFDB Boundary Restriction
This Level 2 qualification is **INVALID** for any files located within an **MFDB child directory**. These environments require **Level 3 (MFDB) Certification**.

## 🏗 Structure
- **Format:** "BEJSON"
- **Format_Version:** "104"
- **Records_Type:** Single-element array identifying the object class.
- **Fields:** Array of name/type dictionaries defining the object attributes.
- **Values:** Array of arrays, where each inner array is a record.

## 🎯 Qualification Mandate
You are **UNQUALIFIED** to modify any BEJSON 104 object unless you have obtained Level 1 Foundational and Level 2 Specialized certification.
